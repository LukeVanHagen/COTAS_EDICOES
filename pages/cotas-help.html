Ajuda - Como Utilizar a Calculadora de Cotas

Esta página explica detalhadamente como utilizar a Calculadora de Cotas, descrevendo cada campo e funcionalidade.

1. Campos de Entrada

  • Nome: Insira o nome do operador ou responsável pela análise.
  • Estação: Indique a estação de trabalho ou local onde os cálculos estão sendo feitos.
  • Tick da Linha: Define o fator de multiplicação para os cálculos das cotas.

2. Estrutura da Tabela

A tabela principal é dividida em três partes:

  • Tabela 1: Contém os valores originais e incrementos.
  • Tabela 2: Permite adicionar incrementos extras.
  • Tabela 3: Exibe o resultado final após os cálculos.

Explicação das Colunas

  • Campo: Nome do parâmetro avaliado (Start, Warning, Exiting, End).
  • Valor Original: Número base antes de qualquer alteração.
  • Valor Secundário: Adiciona um valor extra ao original.
  • Total: Soma dos valores da primeira tabela.
  • Verificação: Indica se o valor total é um múltiplo do Tick da Linha (OK ou KO).
  • Insira o Incremento: Permite adicionar um valor extra.
  • Multiplicador do Tick: Mostra se o incremento respeita o tick da linha.
  • Valor Original (Tabela 3): Mostra novamente o valor original.
  • Incremento: Soma dos incrementos adicionados.
  • Total (Final): Resultado final do cálculo.
  • Verificação Final: Indica se o valor final é válido.

3. Funcionalidade de Validação

Os valores são validados automaticamente, exibindo "OK" se forem múltiplos do Tick ou "KO" caso contrário.


4. Salvamento de Log

Ao clicar no botão "Salvar Log", os dados da análise são baixados em um arquivo JSON nomeado com base na estação, nome do operador e data.


Ajuda - Como Utilizar a Página de Histórico

A página de histórico permite carregar logs salvos anteriormente para revisão e análise.

1. Como Carregar um Arquivo de Log

  • Clique no botão "Selecionar Log".
  • Escolha um arquivo JSON previamente salvo.
  • Os dados do log serão carregados automaticamente nos campos da tabela.

2. Interpretação dos Dados Carregados

Ao carregar um log, os seguintes campos são preenchidos:

  • Nome e Estação: Exibem os dados do operador e do local onde a análise foi realizada.
  • Tick da Linha: Exibe o fator de multiplicação original usado nos cálculos.
  • Tabela de Valores: Exibe os valores originais, incrementos e verificações dos cálculos.

3. Validação Automática

Os valores carregados passam por uma nova verificação para garantir que continuam respeitando as regras de múltiplos do Tick.
